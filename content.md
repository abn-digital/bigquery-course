## Módulo 1: Introducción a SQL
**Tag:** Módulo 1  
**Título:** Introducción a SQL  
**Subtítulo:** El lenguaje universal de los datos. Desde cero hasta escribir tus propias consultas.

### Bloque A — El Lenguaje Básico
- Qué es SQL y por qué importa
- Estructura de una consulta: `SELECT`, `FROM`, `WHERE`
- Tipos de datos: strings, números, fechas
- Cómo leer y entender una tabla

### Bloque B — Filtrar y Agrupar
- Filtrar datos con `WHERE` y operadores lógicos (`AND`, `OR`, `IN`, `BETWEEN`)
- Agrupar con `GROUP BY` y funciones de agregación (`SUM`, `COUNT`, `AVG`)
- Ordenar resultados con `ORDER BY`
- `HAVING` vs `WHERE`: cuándo usar cada uno

### Bloque C — Dimensiones vs. Métricas
- Qué es una dimensión (categorías, fechas, nombres)
- Qué es una métrica (valores numéricos, KPIs)
- Cómo combinarlas para armar reportes útiles
- Ejemplo práctico: reporte de ventas por categoría y mes

### Bloque D — Joins
- Combinar tablas con `JOIN`
- `INNER JOIN`, `LEFT JOIN`: cuándo usar cada uno
- Ejemplo: cruzar tabla de clientes con tabla de transacciones

- Qué es BigQuery y por qué es distinto

- Limitaciones y cuándo conviene materializar

## Módulo 3: Práctica con Tablas Reales
**Tag:** Módulo 3  
**Título:** Práctica con Tablas Reales  
**Subtítulo:** Manos a la obra: creamos tablas, armamos rutinas y dejamos queries corriendo solas.

### Bloque A — Crear una Tabla Externa
- Demo en vivo: conectar un Google Sheet como tabla externa
- Escribir la primera query sobre datos reales
- Validar resultados y entender el schema

### Bloque B — Armar una Rutina para Consolidar Datos
- Qué es una rutina (stored procedure) y para qué sirve
- Crear una rutina que consolide datos de múltiples fuentes
- Parametrizar fechas y filtros

- Qué son las scheduled queries (consultas programadas)
